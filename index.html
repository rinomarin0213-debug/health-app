猫と牛のポーズ (キャット＆カウ)

背骨を動かすことで自律神経が集まる脊髄を刺激し、心身のバランスを整えます。

  1. 四つん這いになり、肩の下に手首、腰の下に膝を置く。
  2. 息を吐きながら、背中を丸め、おへそを覗き込む（猫のポーズ）。
  3. 息を吸いながら、胸を張り、背中を反らせて目線を上へ（牛のポーズ）。
  4. 呼吸に合わせて5〜10回ゆっくり繰り返す。
🛌

チャイルドポーズ

全身の力を抜き、頭を心臓より低くすることで副交感神経を優位にし、高いリラックス効果を得ます。

  1. 正座の姿勢になる。
  2. 息を吐きながら、両手を前に滑らせて上半身を前に倒す。
  3. おでこを床につけ、首や肩の力を抜く。
  4. 背中や腰が広がるのを感じながら、深くゆっくりと呼吸を続ける（1〜3分）。
🌲

木のポーズ

バランスを取ることに意識を向けることで、雑念を払い集中力を高め、心を落ち着かせます。

  1. 両足を揃えてまっすぐ立つ。
  2. 右足に体重をのせ、左足の裏を右足の内もも（またはふくらはぎ、足首）にあてる。（※膝関節は避ける）
  3. 胸の前で合掌し、一点を見つめてバランスをとる。
  4. 余裕があれば両手を上に伸ばし、数回深呼吸する。反対側も行う。
